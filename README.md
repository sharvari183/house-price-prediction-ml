# House Price Prediction using Machine Learning

## Overview

This project predicts house prices using machine learning regression models.

## Models Used

* Linear Regression
* Random Forest Regressor

## Features

* Area
* Bedrooms
* Bathrooms
* Age
* Parking
* Floors
* Location Score

## Visualizations

* Correlation Heatmap
* Model Comparison
* Feature Importance

## Result

Random Forest gave better prediction accuracy.

## Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
